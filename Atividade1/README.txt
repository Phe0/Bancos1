A atividade foi utilizada utilizando python3

Para compilar:

    -acessar a pasta Atividade1
    -no terminal utilizar o comando python3 main.py 
    -inserir os dados referentes ao dono e dos veículos 
    -ao finalizar o programa irá criar uma Pasta Arquivos contendo pastas com a Identidade da Pessoa inserida
    -dentro da pasta da Pessoa, haverá uma descrição dela e uma pasta contendo todos os seus veículos
